## 结论

| 维度 | 优胜方 | 说明 |
|------|--------|------|
| 数据量 | **gweesin** | 多约 6.4 万条地方小机构 |
| 数据新鲜度 | **gweesin** | 每日自动同步 |
| 字段丰富度 | **cwjyz** | 多地址/电话/日期/状态字段 |
| 使用成本 | **gweesin** | 完全免费，MIT |
| 合规性 | **gweesin** | MIT 协议，可商用 |
| 大行数据准确性 | 基本持平 | 差异在 ±1% 以内 |

**推荐：** 核心联行号查询业务直接使用 gweesin/CNAPS CSV，无需付费。若后续需要地址/电话字段，再评估 cwjyz API 费用。
